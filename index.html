Software de escritorio empresarial

Sonalia Desktop centraliza operacion, acceso seguro y automatizacion empresarial.

Sonalia Desktop es una aplicacion de escritorio desarrollada por Software Sonalia para apoyar procesos de gestion empresarial, administracion operativa y envio de notificaciones automaticas. La aplicacion opera localmente en equipos autorizados y utiliza Google OAuth exclusivamente para identificacion y acceso seguro.

Empresa: Software Sonalia Ubicacion: Orizaba, Veracruz, Mexico Contacto: softwaresonalia@gmail.com

Resumen para usuarios y verificacion

  • Aplicacion local de escritorio desarrollada con Electron y Node.js.
  • Uso principal: gestion empresarial y notificaciones operativas.
  • Datos obtenidos por Google OAuth: nombre y correo electronico.
  • No recolecta pagos, ubicacion ni datos personales sensibles.
  • Documentacion legal disponible en Privacidad y Terminos.
Operacion

Control centralizado

Sonalia Desktop concentra funciones operativas y administrativas en una interfaz local para equipos de trabajo autorizados, con enfoque en continuidad, organizacion y seguimiento.

Seguridad

Acceso con Google OAuth

La autenticacion mediante Google se usa para identificar al usuario de forma segura y asociar su cuenta con el perfil correspondiente dentro del sistema.

Automatizacion

Notificaciones y soporte operativo

La aplicacion puede gestionar envios y notificaciones automaticas relacionadas con el uso del sistema y la operacion empresarial.

Uso de datos personales

Software Sonalia aplica un criterio de minimizacion de datos. Para el inicio de sesion con Google solo se solicita la informacion indispensable para identificar al usuario dentro de Sonalia Desktop.

  • Nombre Permite identificar visualmente al usuario dentro del software.
  • Correo electronico Se utiliza para autenticacion, identificacion de cuenta y comunicaciones vinculadas al servicio.
  • Exclusiones expresas No se recolectan datos de pago, ubicacion ni datos sensibles por medio de Google OAuth.
